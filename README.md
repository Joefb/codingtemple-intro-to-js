# Codingtemple Homework: Intro to Javascript

## For this homework assignment

**Add JavaScript to your languages in Codewars:**

- Open codewars.com
- Hover over your rank in the top right (a dropdown will appear)
- Click Training SetUp
- Scroll down and select JavaScript
- Scroll further and Save your Prefrences

**Complete 3 Codewars problems using JS:**

- When selecting a problem, make sure it is set to JavaScript.
- If it says Python in the center of the Title Bar after selecting a problem, click the dropdown and change to JS

**Submitting**
After completing the problem

- Copy BOTH your solution and the prompt for the problem and paste them into vs code js file
- Add it to github and submit the repo link
